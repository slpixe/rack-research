Very deep and voluminous 19-inch server case with 3U and 65cm installation length for professional use in deep server cabinets. This very spacious case offers plenty of room to build a professional server system and supports standard ATX power supplies as well as mini-redundant power supplies, large Extended-ATX motherboards and full-height PCIe/PCI cards.

The case can be used with mini ITX, micro ATX, standard ATX and extended ATX motherboards. Ordinary ATX power supplies can be installed if they have a maximum depth of 20cm. In addition, the installation of mini-redundant power supplies is also possible. The lockable and removable front panel with integrated air filter secures the front-side controls and provides an attractive and professional front finish.

We offer this case also with a matching pre-installed power supply (please see below). If you are interested in a complete server-system based on this case, we can send you a tailor-made offer according to your specification-needs.
