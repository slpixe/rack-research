Product Description

InWin IW-R200-02N is a 2U open-bay short-depth rackmount server chassis supporting ATX and CEB motherboards. Compact 607mm depth design ideal for space-constrained rack environments.

Key Features: Short depth for space-constrained deployments, 2x 5.25" external and 6x 3.5" internal drive bays, 7 low-profile PCIe slots or optional 3 full-height, 28" tool-less & ball-bearing slide rails.

Perfect for space-constrained server deployments, edge computing, and branch office servers where rack depth is limited.
