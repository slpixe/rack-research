## Overview

InWin IW-RS104-07 is a 1U NVMe hybrid rackmount storage server supporting up to 4 hot-swappable 3.5" NVMe/SAS/SATA bays with flexible backplane configurations. Designed for enterprise, data center, and virtualization applications requiring high-performance hybrid storage.
