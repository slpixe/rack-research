Product Description

InWin IW-RA100 is a 1U rackmount chassis supporting ATX motherboards, optimized for space efficiency and HDD deployment. Features outstanding thermal exhaust efficiency designed for appliance use.

Optional riser card for PCI/PCIe expansion and space-efficient airflow design make it suitable for various server applications.
