## Overview

4U Open-Bay Rackmount Chassis with extended storage capacity, supporting EEB/CEB/ATX motherboards, 6x 5.25" external bays, flexible drive bay configurations, and GPU support with optional rear fan module.

## Key Features

- Flexible drive bays configuration for ODD, 2.5" and 3.5" HDD
- Supports easy-swap rear fan x 2 for upgrade
- Compatible with standard PS2, mini-redundant power supply and CRPS
- Expandable storage option with InWin storage kit
- GPU holder and adjustable card retainer supporting 350mm dual-slot PCIe cards x 4
- Optional rear fan module for GPU
- Convertible to tower with chassis stands (optional)
- Lockable front door for data security
- Tool-free cover for easy installation and quick maintenance

## Compatibility

### Industry Standard
- Compatible with EIA-RS310D standard 19" cabinets

### Motherboard Form Factors
- ATX (12" x 9.6")
- CEB (12" x 10.5")
- EEB (12" x 13") - *The bottom 5.25 drive bay space should be reserved for the EEB motherboard installation*

### PSU Support
- PS2 single PSU
- 550W/800W/1200W/1600W 1+1 CRPS 80 Plus Platinum
