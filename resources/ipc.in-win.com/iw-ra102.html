Product Description

InWin IW-RA102 is a compact 1U rackmount chassis designed for space-sensitive server deployments. It features dual hot-swap 2.5" drive bays for quick maintenance and supports ATX/Micro-ATX/Mini-ITX motherboards with flexible power supply options.

Outstanding thermal exhaust efficiency and flexible PSU options make it suitable for custom server builds and enterprise deployments.
