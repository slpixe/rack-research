Product Description

InWin IW-RS104-07 is a 1U NVMe hybrid rackmount storage server supporting up to 4 hot-swappable 3.5" NVMe/SAS/SATA bays. Features flexible backplane configurations with OCuLink and Slimline connectors for PCIe Gen 4.0 NVMe support.

Key Features: Hybrid storage solution with NVMe/SAS/SATA interchangeability, 6 high-speed easy-swap fans at 21,500 RPM, tool-less design for quick maintenance, and optional redundant power supply for high availability.

Designed for enterprise, data center, and virtualization applications requiring high-performance hybrid storage in a compact 1U form factor.
