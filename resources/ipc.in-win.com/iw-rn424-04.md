## Overview

4U Dual Node Rackmount Chassis with 24x hot-swap 3.5" drive bays, two independent hot-swappable system nodes, SAS3 12Gb/s expander support, and high-efficiency CRPS power supply.

## Key Features

- Optimized performance for each node
- Easy maintenance and high availability with two hot-swappable system nodes
- Supports SAS 12Gbps via single expander for each node
- Tool-less design for 3.5" HDD carrier
- Compatible with standard form factor motherboard
- Supports tool-less slide rail
- Supports high-efficiency CRPS power supply
