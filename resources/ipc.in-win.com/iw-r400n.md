## Overview

InWin IW-R400N is a 4U open-bay rackmount chassis with flexible drive bay configuration supporting ODD, 2.5" and 3.5" HDDs. Features tool-free cover, lockable front door, and convertible tower design with optional stands.

## Key Features

- Flexible drive bays configuration for ODD, 2.5" and 3.5" HDD
- Supports easy-swap rear fan x 2 for upgrade
- Compatible with standard PS2 or mini-redundant power supply
- Expandable storage option with InWin storage kits
- GPU holder and adjustable card retainer supporting 350mm dual-slot PCIe cards x 4
- Convertible to tower with chassis stands (optional)
- Lockable front door for data security
- Tool-free cover for easy installation and quick maintenance
