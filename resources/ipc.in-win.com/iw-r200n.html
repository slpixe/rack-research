Product Description

InWin IW-R200N is a 2U open-bay middle-depth rackmount server chassis supporting ATX and CEB motherboards. Features flexible storage configuration with 2x 5.25" external and 6x 3.5" internal drive bays.

Key Features: Open-bay design for general server deployments, supports 7 low-profile PCIe slots or optional 3 full-height slots, 28" tool-less & ball-bearing slide rails, optional intrusion switch for security.

Ideal for general purpose servers, storage servers, and workstations requiring flexible configuration options and multiple drive bays.
