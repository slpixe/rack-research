Product Description

InWin IW-RS108-07 is a 1U NVMe hybrid rackmount storage server supporting up to 8 hot-swappable U.2 NVMe/SAS/SATA disks. Features flexible hybrid configuration with OCuLink connectors for maximum NVMe SSD deployment.

Key Features: High-density storage with 8 hot-swappable bays, PCIe 4.0 support, tool-less maintenance, extensive airflow design, and compatibility with Intel Xeon and AMD EPYC platforms.

Handles both 3.5" HDDs and 2.5" SSDs in various thicknesses, making it ideal for flexible storage configurations in enterprise and data center environments.
