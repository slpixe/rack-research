## Overview

4U NVMe Hybrid Rackmount Chassis - A hybrid storage server for mixed-use workloads. Designed for large amounts of data, real-time analytics, high-speed data streaming, and mainstream enterprise environments. The 36-bay high-density drawer configuration maximizes storage capacity in a space-saving 4U form factor.

## Key Features

- Intel® Xeon Scalable & AMD EPYC family compliant
- Backplanes with Mini-SAS HD, OCuLink, or SlimSAS connectors (compatible with PCIe 4.0)
- End-to-end 12Gbps SAS host, expander, and disk support
- 2.5"/3.5" HDD trays x 36 (optional NVMe trays x 12, distinguishes U.2 NVMe SSDs)
- 2 x 2.5" SAS/SATA SSDs for system disks at the rear of the chassis
- Highly efficient 1200W, 80 Plus® Platinum CRPS (optional)
- High-speed cooling fans for excellent thermal performance
- 4U High-density drawer configuration design for space-saving
