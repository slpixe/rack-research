## Overview

The IW-RG650A is a 6.5U GPU rackmount chassis specially designed for AI, Deep Learning, and HPC applications. It features an innovative dual-chamber design with a 4.5U upper chamber for GPUs and a 2U lower chamber for system components, providing independent heat dissipation and optimized cooling performance.

---

## Key Features

- GPU Server Specially Designed for AI, Deep Learning, and HPC
- 6.5U Dual Chamber Design: 4.5U Chamber for GPU and 2U Chamber for M/B
- Supports 18 FHFL PCIe Slots with 4 PCIe Riser Cables Included for Multi-GPU Installation
- Separate Dual Chambers for Optimized, Independent Heat Dissipation
- 9 Hot-Swap 80x38 mm 12000rpm PWM Fans for Excellent Cooling Performance
- 80 PLUS® Titanium CRPS 3200W PSU with 4 x 12V-2x6 pin Connectors Fulfill GPU Wattage Needs

| Fan Speed | 12000 RPM |
| Installation | Pre-installed |

### Airflow Design
- Dual-chamber design provides separate airflow tunnels for GPUs and system components
- Independent heat dissipation prevents thermal interference between chambers
- 6 fans cool the GPU chamber, 3 fans cool the system component chamber
