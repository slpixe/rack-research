Product Description

InWin IW-RL200 is a 2U liquid-cooled rackmount chassis designed for edge computing applications. Features dual cooling design supporting 240mm AIO liquid cooler for CPU and air cooling for GPU simultaneously.

Key Features: Supports up to EEB motherboards, 1+1 redundant 1200W PSU, 3 full-height PCIe slots, tool-less top cover for maintenance, and optional intrusion switch for security.

Ideal for edge computing, retail, smart factory, digital banking, and industrial automation applications where both high performance and reliable cooling are required.
