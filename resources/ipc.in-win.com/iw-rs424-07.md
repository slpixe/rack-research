## Overview

4U NVMe Hybrid Rackmount Chassis - A comprehensive NVMe storage server solution for enterprise. Scalable, high-performance NVMe hybrid storage architecture compliant with Intel® Xeon Scalable processors & AMD EPYC family, delivering extreme performance and ultra-low latency for top-tier enterprise storage.

## Key Features

- Intel® Xeon Scalable & AMD EPYC family compliant
- Backplanes with Mini-SAS HD, OCuLink or SlimSAS connectors (compatible with PCIe 4.0)
- End-to-end 12Gbps SAS host, expander and disk support
- 2.5"/3.5" hot-swap HDD trays x 24 (optional NVMe trays x 8, distinguishes U.2 NVMe SSDs)
- 2 x 2.5" SAS/SATA SSDs for system disks at the rear of the chassis
- Highly efficient 550W/800W/1200W/1600W, 80 Plus® Platinum CRPS (optional)
- High-speed cooling fans for excellent thermal performance
- GPU-optimized design allows for multiple dual-slot GPUs
