## Overview

4U ATX rackmount chassis designed for server environments, data centers, industrial PCs, and SMBs. Features 360mm liquid cooling compatibility (bracket included), modern USB Type-C connectivity, key lock security, anti-intrusion system, and includes four legs for vertical mounting. Includes 2x 80mm rear fans.
