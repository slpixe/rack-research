# RM32

3U rackmount server chassis supporting 4-slot high-end graphics cards and 360mm radiators

| Attribute | Value |
| --- | --- |
| Model No. | SST-RM32 |
| Chassis Form Factor | 3U |
| Material | Steel |
| Motherboard | SSI-EEB, SSI-CEB, Extended ATX, ATX, Micro-ATX, Mini-ITX |
| Drive bay | Internal: 3.5"x2 or 2.5"x2, 2.5"x3 |
| Cooling system | Front: 120mm x 3; Rear I/O: 60mm x 2 |
| Radiator support | Front: 120mm / 240mm / 360mm (Fans + radiators up to 80mm thickness) |
| Limitation of CPU cooler | 110mm; 86mm (If the CPU Cooler area is located below the riser cable) |
| Expansion slot | 4 |
| Limitation of expansion card | Length: 360mm; Width: 177mm (with SFX-L PSU), 162mm (with SilverStone 2U CRPS redundant PSU) |
| Power supply | SFX/SFX-L / SilverStone 2U CRPS Redundant |
| Limitation of PSU | 250mm (Removal of 3.5" drive cage eliminates PSU depth limitation) |
| Front I/O port | USB Type-C x 1, USB 3.0 x 2 |
| LED Indicators | Power status x 1: Blue; HDD activity x 1: White; LAN activity x 2: Green |
| Dimension | 438mm (W) x 132mm (H) x 488mm (D) |
| Remark | Side switch for chassis intrusion security |
