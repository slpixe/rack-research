# S34T

The S34T is a compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX motherboards and features a very small footprint, making it ideal for specialized industrial applications requiring legacy port support.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S34T_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX (max. 170 x 170mm - 6.7"x6.7") |
| **Case Dimensions (H x W x D)** | 303 x 140 x 230mm |
| **Drive Bays** | 2.5" internal: 1<br>3.5" or 2.5" internal: 1 (either 1x 3.5" or 2x 2.5") |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 2x 80mm (included) |
| **Maximum CPU Cooler Height** | 70mm |
| **Maximum Graphics Card Length** | 180mm |
| **Power Supply** | Flex-ATX |
| **PCI Slots** | 2x Full-Size |

## Features

- Two 80mm fans included
- Exchangeable dust filter
- Cutouts for 4x DB9/DB15 ports
- Cutout for 1x Dual-USB port
- Cutout for 1x 50-pin HD-SCSI port
- Two Full-Size PCI-slots
- Perforated slot brackets
- Supports wall mounting (4 stands included)
