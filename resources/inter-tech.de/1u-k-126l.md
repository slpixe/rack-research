# IPC 1U K-126L

The 1U K-126L is the perfect rack for a small and cost-effective server system. Its compact design and its small installation depth of 250mm for ITX boards allows this case to fit into nearly every rack.

The 1U K-126L provides internal space for optional 1x 3.5" or 2x 2.5" hard disks. Two serial 40mm fans, which are internally mounted behind the front panel, ensure a reliable and optimal cooling. Further you can two USB 2.0 connectors in the front of the case next to the controls.

## Specifications

| Feature | Details |
| :--- | :--- |
| **Form Factor** | 1U |
| **Motherboard Support** | Mini ITX |
| **Dimensions (H x W x D)** | 45 x 485 x 285mm (Chassis depth: 250mm) |
| **Drive Bays** | 1x 3.5" internal or 2x 2.5" internal |
| **Front Connectors** | 2x USB 2.0 |
| **Cooling System** | 2x 40mm fans included (serial, Molex connector) |
| **Expansion Slots** | None (standard) |
| **Power Supply Support** | FlexATX PSU (brackets included) |
| **Max CPU Cooler Height** | 32mm |
| **Max Graphic Card Length** | 170mm |
| **Material** | 1.0-1.2mm Steel |
| **Features** | Only 250mm Depth, I/O Shield (Universal) included |
