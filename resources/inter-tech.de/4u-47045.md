# 4U-47045

The 4U-47045 is a versatile 4U server case with many expansion options and is suitable for use in a 19" rack or as a stand-alone tower.

[Product Page](https://www.inter-tech.de/productdetails-149/4U-47045_EN.html)

## Specifications

| Attribute | Value |
| --- | --- |
| **Features** | Modular airflow cage system, Case-open switch, Installation options for multiple fans, Graphics card holder, Can be converted to a tower case |
| **Mainboard Support** | Mini-ITX, Micro-ATX, ATX, E-ATX (max. 305 x 330 mm (12" x 13")) |
| **Case Dimensions (W x D x H)** | 480 x 485 x 175mm |
| **Cage Dimensions (W x D x H)** | 430 x 445 x 175mm |
| **5.25" Drive Bays (External)** | 3 (plus 1 Slim DVD) |
| **3.5" Drive Bays (Internal)** | 4 |

| **Front Connectors** | 2x USB 3.2 Gen1 |
| **Power Supply Support** | ATX, Mini-Redundant PS/2 |
| **CPU Cooler Height (Max)** | 145 mm |
| **Graphics Card Length (Max)** | 330 mm |
| **Fan Support** | 2x 80mm rear, 1x 80mm in the hard drive cage, 1x 120mm in front door |
