# S34N

The S34N is a compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX and Micro-ATX motherboards and features a very small footprint, making it ideal for specialized industrial applications requiring multiple serial port options.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S34N_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX, Micro-ATX (max. 244 x 190mm - 9.6"x7.5") |
| **Case Dimensions (H x W x D)** | 303 x 140 x 230mm |
| **Drive Bays** | 2.5" internal: 1<br>3.5" or 2.5" internal: 1 (either 1x 3.5" or 2x 2.5") |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 2x 80mm (included) |
| **Maximum CPU Cooler Height** | 70mm |
| **Maximum Graphics Card Length** | 180mm |
| **Power Supply** | Flex-ATX |
| **PCI Slots** | 4x Full-Size |

## Features

- Two 80mm fans included
- Exchangeable dust filter
- Cutouts for 4x DB9/DB15 ports
- Cutout for 1x Dual-USB port
- Four Full-Size PCI-slots
- Perforated slot brackets
- Supports wall mounting (4 stands included)
