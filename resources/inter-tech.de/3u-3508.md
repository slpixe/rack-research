# 3U-3508

The IPC 3U-3508 is a perfect case for a cost-efficient Mini Server System. Using high-level material combined with excellent workmanship we created a variable and reliable industrial case.

So the 3U-3508 supports ATX power supplies, up to eight 2.5"/3.5" Hot-Plug hard disks, two internal 2.5" devices, two external 5.25" devices and mainboards up to ATX size.

Furthermore the case has two 80mm fans and two USB 3.2 Gen1 front ports.
