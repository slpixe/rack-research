# IPC 2U-K-240L

The 2U-K-240L is a perfect case for a cost-efficient Mini Server System. Using high-level material combined with excellent workmanship we created a variable and reliable industrial case.

So the 2U-K240L supports ATX power supplies with 80mm fan, up to four 3,5“ internal devices, two internal 2,5“ devices, one external 5,25“ device and mainboards up to Micro-ATX size.

Furthermore the case has two 80mm fans and two USB 3.2 Gen1 front ports.

## Specifications

| Feature | Details |
| :--- | :--- |
| **Form Factor** | 2U |
| **Motherboard Support** | Mini-ITX, Micro-ATX (max. 244 x 244mm - 9.6" x 9.6") |
| **Dimensions (H x W x D)** | 89 x 482 x 433mm |
| **External Drive Bays** | 1x 5.25" |
| **Internal Drive Bays** | 4x 3.5", 2x 2.5" |
| **Front Connectors** | 2x USB 3.2 Gen1 |
| **Cooling System (Front)** | 2x 80mm (included, Molex connector) |
| **Cooling System (Rear)** | 2x 60mm (optional) |

| **Max CPU Cooler Height** | 60mm |
| **Max Graphic Card Length** | 244mm (Low-Profile) |
| **Features** | Internal HDD cages removable, Aluminium Front |
