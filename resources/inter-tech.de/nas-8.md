# NAS-8

The NAS-8 is a stand case made for use as NAS-System.

Eight removal trays, connected to two 12G backplanes and a well thought out ventilation concept create optimal conditions for a professional NAS system.

Two build-in 120mm fans, an USB A 3.2Gen1 and an USB C 3.2Gen1 connection round up this treasure.
