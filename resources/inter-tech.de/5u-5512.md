# 5U-5512

The IPC 5U-5512 is a perfect case for a cost-efficient Mini Server System. Using high-level material combined with excellent workmanship we created a variable and reliable industrial case.

The 5U-5512 supports ATX or PS/2 redundant power supplies, up to twelve 2.5"/3.5" Hot-Plug hard disks, five internal 2.5" devices, two external 5.25" devices and mainboards up to E-ATX size. Furthermore the case has two 80mm fans and two USB 3.2 Gen1 front ports.

## Specifications

| Attribute | Details |
|-----------|---------|
| **Motherboard** | ITX, Micro-ATX, ATX, E-ATX, SSI EEB (E-ATX and SSI Mainboards must not have parts in the front area with height of more than 3 cm) |
| **CPU Cooler Height (max.)** | 105mm |
| **Graphics Card Length (max.)** | 250mm |
| **Drive Bays** | 12x 2.5" or 3.5" external (Hot-Swap)<br>2x 5.25" external<br>5x 2.5" internal |
| **Front Connectors** | 2x USB 3.2 Gen1 |
| **PSU Support** | ATX or Mini-Redundant PS/2 |
| **Fan Mounting Options** | Mounted: 3x 80mm<br>Rear: 2x 80mm (optional) |
| **Backplane** | 3x Backplanes (4x SATA each), 6Gb/s, 2x Molex per backplane |
| **Dimensions (H/W/D)** | 225 x 481 x 505mm |
| **Weight (Net)** | 13.56kg |
| **Scope of Delivery** | Case, Power supply mounting for ATX (pre-installed), Screws |
| **Features** | 12 Hot-Swap HDD frames, HDD frames lockable, Serial three 80mm fans, Perforated slot brackets |

## Backplane Details
- Quantity Backplanes: 3
- Quantity hard disks: 12
- Transfer speed: 6Gb/s
- Data connection: 4x SATA per Backplane
- Electrical connection: 2x Molex per Backplane
