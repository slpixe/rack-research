# S34

The S34 is a compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX and Micro-ATX motherboards and features a small footprint, making it ideal for specialized industrial applications requiring multiple serial and USB port options.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S34_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX, Micro-ATX (max. 244 x 244mm - 9.6"x9.6") |
| **Case Dimensions (H x W x D)** | 303 x 195 x 330mm |
| **Drive Bays** | 3.5" or 2.5" internal: 1 |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 1x 80mm (included) |
| **Maximum CPU Cooler Height** | 125mm |
| **Maximum Graphics Card Length** | 280mm |
| **Power Supply** | Flex-ATX |
| **PCI Slots** | 4x Full-Size |

## Features

- One 80mm fan included
- Exchangeable dust filter
- Cutouts for 4x DB9/DB15 ports
- Cutouts for 2x Dual-USB ports
- Four Full-Size PCI-slots
- Perforated slot brackets
- Supports wall mounting (4 stands included)
