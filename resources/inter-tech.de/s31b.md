# S31B

The S31B is a compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX motherboards and features a small footprint, making it ideal for specialized industrial applications requiring multiple serial port options and a dual-PCI slot.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S31B_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX (max. 170 x 170mm - 6.7"x6.7") |
| **Case Dimensions (H x W x D)** | 270 x 230 x 167mm |
| **Drive Bays** | 3.5" or 2.5" internal: 1 (1x 3.5" or 2x 2.5") |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 1x 80mm (included) |
| **Maximum CPU Cooler Height** | 125mm |
| **Maximum Graphics Card Length** | 160mm |
| **Power Supply** | Flex-ATX |
| **PCI Slots** | 2x (Dual-PCI Slot) |

## Features

- One 80mm fan included
- Cutouts for 6x DB9/DB15 ports
- One Dual-PCI Slot
- Perforated slot bracket
- Supports wall mounting (4 stands included)
- Includes bracket to close power supply cutoff and angle for FlexATX power supply
