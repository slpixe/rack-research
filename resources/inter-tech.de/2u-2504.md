# 2U-2504

The IPC 2U-2504 is a perfect case for a cost-efficient Mini Server System. Using high-level material combined with excellent workmanship we created a variable and reliable industrial case.

The 2U-2504 supports ATX power supplies (only with 80mm fan), up to four 2.5"/3.5" Hot-Plug hard disks, two internal 2.5" devices, one external 5.25" devices and Mini-ITX and mATX mainboards.

Furthermore the case has two 60mm fans and two USB 3.2 Gen1 front ports.

## Specifications

| Feature | Details |
| :--- | :--- |
| **Motherboard** | Mini-ITX, Micro-ATX (max. 244 x 244mm) |
| **Drive Bays** | 4x 3.5" or 2.5" external (Hot-Swap)<br>1x 5.25" external<br>2x 2.5" internal |
| **Front Connectors** | 2x USB 3.2 Gen1 |
| **Cooling System** | Internal: 2x 60mm (serial, Molex connector) |
| **Controls & LED** | Controls: Power, Reset<br>LED: Power, HDD, 2x Network, Alert |
| **Power Supply** | ATX with 80mm fan |
| **Material** | Steel |
| **Dimensions (H/W/D)** | 89 x 480 x 527mm |
| **Weight** | Net: 7.71kg, Gross: 10.53kg |
| **Backplane** | 2x Backplanes (2 drives each)<br>SATA 6 Gb/s<br>Data connection: 2x SATA per backplane |
| **Features** | 4x Low-Profile slot brackets (Rear)<br>Hot-Swap frames lockable |
