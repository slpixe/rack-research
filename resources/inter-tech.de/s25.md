# S25

The S25 is a low-profile industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX motherboards and features a very slim height of 89mm, making it ideal for space-constrained industrial applications requiring extensive serial port connectivity and low-profile expansion.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S-25_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX (max. 170 x 170mm - 6.7"x6.7") |
| **Case Dimensions (H x W x D)** | 89 x 315 x 227mm |
| **Drive Bays** | 3.5" or 2.5" internal: 1 (1x 3.5" or 2x 2.5") |
| **Front Connectors** | USB 3.0: 2 |
| **Fan Mounting Options** | Front: 1x 80mm (included) |
| **Maximum CPU Cooler Height** | 45mm (depends on hardware) |
| **Maximum Graphics Card Length** | 180mm |
| **Power Supply** | Flex-ATX |
| **PCI Slots** | 1x Low-Profile (Dual PCI-Slot support) |

## Features

- One 80mm fan included
- Cutouts for 8x DB9/DB15 ports in the rear
- Dual PCI-Slot (Low-Profile)
- Perforated slot brackets
- Supports wall mounting (4 stands included)
