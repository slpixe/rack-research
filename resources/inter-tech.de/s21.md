# S21

The S21 is an ultra-compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX motherboards and features a very small footprint, making it ideal for space-constrained industrial applications.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S-21_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX (max. 170 x 170mm - 6.7"x6.7") |
| **Case Dimensions (H x W x D)** | 98 x 225 x 210mm |
| **Drive Bays** | 3.5" internal: 1<br>2.5" internal: 2 |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 1x 80mm (included) |
| **PCI Slots** | 1x Full-size (Riser card necessary) |

## Features

- One 80mm fan included
- Cutouts for 2x DB9/DB15 ports
- Single PCI-Slot (Riser card necessary)
- Perforated slot bracket
- Supports wall mounting (4 stands included)
