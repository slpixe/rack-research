# S37

The S37 is a compact industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX, Micro-ATX, and ATX motherboards and features a shallow depth, making it suitable for space-constrained industrial environments.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S37_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX, Micro-ATX, ATX (max. 305 x 244mm - 12"x9.6") |
| **Case Dimensions (H x W x D)** | 360 x 235 x 330mm |
| **Drive Bays** | 2.5" internal: 1<br>3.5" or 2.5" internal: 1 |
| **Front Connectors** | USB 2.0: 2 |
| **Fan Mounting Options** | Front: 2x 80mm (included) |
| **Maximum CPU Cooler Height** | 155mm |
| **Maximum Graphics Card Length** | 280mm |
| **Power Supply** | ATX |
| **PCI Slots** | 7x Full-Size |

## Features

- Two 80mm fans included
- Exchangeable dust filter
- Cutouts for 2x DB9/DB15 ports
- Seven Full-Size PCI-slots
- Perforated slot brackets
- Supports wall mounting (4 stands included)
