# IP-40

The IP-40 is a compact, half-height aluminum case specifically designed for Thin Mini-ITX motherboards. It is optimized for passive cooling and supports CPUs with a TDP of up to 70W when paired with the optional IP-3 cooler.

## Specifications

| Attribute | Value |
|-----------|-------|
| Motherboard | Thin Mini-ITX (max. 170mm x 170mm) |
| Dimensions (H/W/D) | 58 x 235 x 200mm |
| Material | Aluminum |
| Drive Bays | 2.5" internal: 1x |
| Front Connectors | USB 3.0: 2x |
| Power Supply | Via motherboard (Thin Mini-ITX) |

## Features

- Compact half-height design
- Integrated cooling fins for heat dissipation
- Wall mounting via standard rails (included)
- VESA mount compatible (optional)
- Optimized for passive cooling systems
