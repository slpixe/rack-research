# S38

The S38 is a versatile industrial case designed for cost-efficient Mini Server Systems. It supports Mini-ITX, Micro-ATX, and ATX motherboards and offers a range of external and internal drive bays, along with specialized connectivity options for industrial applications.

[Go to Inter-Tech website](https://www.inter-tech.de/productdetails-138/S38_EN.html)

## Specifications

| Attribute | Value |
| :--- | :--- |
| **Form Factor** | Industrial Case (Desktop/Wallmount) |
| **Mainboard** | Mini-ITX, Micro-ATX, ATX (max. 305 x 244mm - 12"x9.6") |
| **Case Dimensions (H x W x D)** | 330 x 205 x 405mm |
| **Drive Bays** | 5.25" external: 1<br>3.5" external: 2<br>3.5" internal: 1<br>2.5" internal: 1 |
| **Front Connectors** | USB 2.0: 2<br>DC-Output: 1 (Power switch looped through) |
| **Fan Mounting Options** | Front: 1x 120mm (included) |
| **Maximum CPU Cooler Height** | 85mm |
| **Maximum Graphics Card Length** | 330mm |
| **Power Supply** | ATX |
| **PCI Slots** | 7x Full-Size |

## Features

- One 120mm fan included
- Exchangeable dust filter
- Removable and decoupled HDD cage
- Separately decoupled internal 3.5" device
- Internal speaker included
- Cutouts for 5x DB9/DB15 ports
- Cutout for 1x DB25/Printer port
- Screwed slot brackets
- Supports wall mounting (4 stands included)
