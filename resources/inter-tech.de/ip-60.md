# IP-60

The IP-60 is a compact Mini-ITX case made of aluminum, designed for passive cooling. It features integrated cooling fins and supports CPUs with a TDP of up to 70W when used with the optional IP-3 cooler.

## Specifications

| Attribute | Value |
|-----------|-------|
| Motherboard | Mini-ITX (max. 170mm x 170mm) |
| Dimensions (H/W/D) | 77 x 210 x 240mm |
| Material | Aluminum |
| Drive Bays | 2.5" internal: 2x |
| Front Connectors | USB 3.0: 2x |
| CPU Cooler Height (max.) | 43mm |
| Power Supply | Mini-ITX PSU (120W included) |

## Features

- Integrated cooling fins for heat dissipation
- Wall mounting via standard rails (included)
- 1x Serial port cutout (back)
- 1x WLAN cutout (back)
- Optional USB 3.2 Type C socket
- Four feet included for standing operation
- VESA mount compatible (optional)
