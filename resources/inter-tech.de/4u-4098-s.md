# IPC 4U 4098-S

The 4U 4098-S - Classic 19" Rack with a lot of space for different drives.

You can remove the HDD cages with a few steps for easy mounting of internal devices.

You can mount 4 x 2.5" Hard drives on a crossbeam inside the case.

The lockable doors in front of the slots prevent, unauthorized access.

Integrated air filters ensure clean air and a flawless cooling.

## Specifications

| Feature | Specification |
| --- | --- |
| **Motherboard** | Mini ITX, mATX, μATX, ATX (max. 305mm x 300mm) |
| **Drive Bays** | 5.25" external: 6x¹; 3.5" internal: 5x¹; 2.5" internal: 6x² |
| **Front Connectors** | USB 3.0: 2x |
| **Cooling System** | Rear: 2x 80mm (included) |
| **Controls & LED** | Power-Button, Reset-Button; Power LED, HDD LED |
| **Dimensions (H/W/D)** | 178 x 485 x 520mm |
| **Material** | 1.0-1.2mm Steel |
| **Max CPU Cooler Height** | 130mm³ |
| **Max Graphics Card Length** | 290mm⁴ |
| **Features** | Air filter, Removable HDD cages, 7x screwed brackets (backside), Lockable front doors, USB 3.0 internal as block connector |

¹ can vary by using several mounting options
² max height 4x 9.5mm and 2x 15mm
