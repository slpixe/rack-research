# Sliger CX3171i XL

- **Manufacturer:** Sliger
- **Product Page:** [CX3171i XL](https://sliger.com/products/cx3171i_xl)
- **Type:** 3U Rackmount
- **Price:** $339.00 (Base Price)

## Dimensions
- **Depth:** 17" (431.8mm)
- **Width:** 17" (431.8mm)
- **Height:** 5.22" (132.59mm)
- **Weight:** 17.75 lbs (8.05 kg)

## Motherboard Support
- Mini-ITX
- Mini-DTX
- Deep Mini-ITX
- FlexATX
- MicroATX
- Deep MicroATX
- ATX
- EATX
- SSI-CEB
- SSI-EEB
- XL-ATX/EATX (up to 12.8" x 13")

## Power Supply Support
- SFX
- SFX-L

## Cooling Support
- **CPU Air Cooler:** Max Height 110mm
- **Liquid Cooling:**
  - 1x 120mm or 1x 240mm radiator
  - Max radiator length: 277mm (240mm)
- **Fans:**
  - Front: 2x 120mm (up to 38mm thick)
  - Rear: 1x 80mm fan (optional PCIe slot to fan adapters)

## PCIe / GPU Support
- **Slots:** 3x Full Height Horizontal PCIe slots (via PCIe 5.0 16x riser)
- **Max GPU Width:** 72mm (3.5-slot)
- **Max GPU Height:** 150mm
- **Max GPU Length:**
  - 375mm (with 2x 25mm front fans)
  - 347mm (with 240mm AIO + 2x 25mm fans)

## Storage
- 2x 2.5" drives
- 1x External 5.25" bay (can hold 2x 2.5" and 1x 3.5" via filler blank)

## Features
- PCIe 5.0 16x riser for horizontal GPU mounting
- Rear support bridge to prevent GPU sag
- Washable/reusable dust filter included
- Planned availability through 2032
- Made in USA
