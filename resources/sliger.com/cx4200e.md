# Sliger CX4200e

- **Price:** $299.00
- **Form Factor:** 4U
- **Depth:** 508mm (20")
- **Motherboard Support:** Mini-ITX, Mini-DTX, FlexATX, MicroATX, ATX, SSI-CEB, EATX (up to 12" x 10.5"), SSI-EEB (up to 12" x 13"), XL-ATX (up to 12.8" x 13")
- **Storage:**
    - 2x 5.25" External Bays
    - 2x 2.5" Internal SSD mounts (rear, above PSU)
    - 5.25" bay filler blank provides mounting for 2x 2.5" SSD and 1x 3.5" HDD
- **Cooling:**
    - **Air Cooling:** Max height 153mm
    - **Liquid Cooling:** Up to 1x 240mm AIO (max length 277mm)
    - **Front Fans:** 2x 120mm (up to 38mm thick) or 4x 120mm (push/pull with 240mm radiator)
    - **Rear Fans:** Optional 1x 120mm bracket
- **GPU Support:**
    - 8x Full Height PCIe slots
    - Max Height: 158mm
    - Max Length (fans only): 450mm
    - Max Length (240mm AIO + 2x 25mm fans): 422mm
    - Max Length (240mm AIO + 4x 25mm fans): 398mm
- **Power Supply:** ATX PSU (max length 268mm) or CRPS Redundant PSU
- **Front I/O:** 1x Power Button with LED, 2x USB 3.0 Type A, 1x USB 3.2 Type C (10Gbps)
- **Dimensions:** 508mm x 431.8mm x 176.78mm (20" x 17" x 6.96")
