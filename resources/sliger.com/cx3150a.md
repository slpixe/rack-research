# CX3150a

The CX3150a is a 15" deep 3U rackmount chassis designed for high-performance systems with extensive liquid cooling support. It supports up to a 360mm AIO and full-size GPUs.

## Pricing

- Base Price: $219.00 USD
- Shipping: Free Shipping (US Only)

## Dimensions

| Attribute | Value |
|-----------|-------|
| Outer Dimensions (D x W x H) | 381.00mm x 431.80mm x 132.59mm (15" x 17" x 5.22") |
| Weight | 13.75 lbs (6.24 kg) |
| Volume | 21.90 L |

## Compatibility

| Attribute | Value |
|-----------|-------|
| Motherboard | Mini-ITX, Mini-DTX, Deep Mini-ITX, FlexATX, MicroATX, Deep MicroATX, ATX, EATX (up to 10.5" deep), SSI-CEB |
| Power Supply | SFX or SFX-L |
| GPU Length | 324mm (with 25mm fans), 296mm (with 360mm AIO + 25mm fans) |
| GPU Height | 115mm |
| PCIe Slots | 8x Full Height |

## Cooling

| Attribute | Value |
|-----------|-------|
| CPU Air Cooler Height | 110mm (max) |
| Liquid Cooling | 1x 360mm AIO, 1x 240mm AIO, or 2x 120mm AIOs |
| Front Fans | 3x 120mm (up to 38mm thick) |
| Rear Fans | 1x 120mm |
| Dust Filter | 1x washable/reusable 30 ppi foam filter included |

## Storage & I/O

| Attribute | Value |
|-----------|-------|
| Internal 2.5" Bays | 1x (up to 15mm thick) |
| Front I/O | 2x USB 3.0 Type A, 1x USB 3.2 Type C (10Gbps) |
| Controls | Momentary Power Switch with White LED Ring |

- Rails: General Devices C300-S Series (optional)
- Rail Compatibility: iStarUSA TC Rails natively compatible
