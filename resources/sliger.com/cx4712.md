# Sliger CX4712

- **Price:** $439.00
- **Form Factor:** 4U
- **Depth:** 635mm (25")
- **Motherboard Support:** Mini-ITX, Mini-DTX, FlexATX, MicroATX, ATX, SSI-CEB, EATX (up to 12" x 10.5")
- **Storage:**
    - 10x 3.5" Trayless Hot-Swap SATA/SAS HDD Bays
    - 2x 5.25" External Bays
    - 6x 2.5" Internal SSD mounts total
    - 5.25" bay filler blank provides mounting for 2x 2.5" SSD and 1x 3.5" HDD
- **Cooling:**
    - **Air Cooling:** Max height 153mm
    - **Liquid Cooling:** Up to 1x 360mm AIO (max length 395mm) or 1x 240mm AIO (max length 277mm)
    - **Mid-Chassis Fans:** 3x 120mm (up to 38mm thick) or 6x 120mm (push/pull with 360mm radiator)
    - **Rear Fans:** Optional 1x 120mm bracket
    - **Note:** Push/pull configuration interferes with internal drive cage.
- **GPU Support:**
    - 8x Full Height PCIe slots
    - Max Height: 158mm
    - Max Length (3x 120mm fans): 337mm
    - Max Length (360mm AIO + 3x 120mm fans): 309mm
    - Max Length (360mm AIO + 6x 120mm fans): 285mm
- **Power Supply:** ATX PSU (max length 268mm) or CRPS Redundant PSU
- **Front I/O:** 1x Power Button with LED, 2x USB 3.0 Type A, 1x USB 3.2 Type C (10Gbps)
- **Dimensions:** 635mm x 431.8mm x 176.78mm (25" x 17" x 6.96")
